5. Reactor Parameters¶

This document lists all of the Reactor Parameters that are provided by the ARMI Framework and included plugins.

Reactor Parameters from Framework¶

Name

Description

Number of region-density equilibrium iterations

int

cycle

current cycle of the simulation

cycleLength

The cycle length of the reactor while power is being produced

EFP days

availabilityFactor

Availability factor of the plant. This is the fraction of the time that the plant is operating.

fraction

capacityFactor

The fraction of power produced by the plant this cycle over the full-power, 100% uptime potential of the plant.

fraction

$/kWh

MT

kgSWU
