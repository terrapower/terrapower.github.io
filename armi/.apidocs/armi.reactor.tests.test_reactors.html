Loads a test reactor. Can be used in other test modules.

Parameters

  • inputFilePath (str) – Path to the directory of the input file.

  • customSettings (dict with str keys and values of any type) – For each key in customSettings, the cs which is loaded from the armiRun.yaml will be overwritten to the value given in customSettings for that key.

  • inputFileName (str, default="armiRun.yaml") – Name of the input file to run.

Returns

  • o (Operator)

  • r (Reactor)
